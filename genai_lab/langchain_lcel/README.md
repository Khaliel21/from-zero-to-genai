# ?? LangChain LCEL (LangChain Expression Language)

This folder contains notebooks that demonstrate how to build LLM-powered applications using the LCEL paradigm.

## Contents

- `06_LCEL1.ipynb` to `08_LCEL1.ipynb` – Practical examples of using LCEL with components like `RunnableMap`, `RunnableSequence`, etc.

## Purpose

To showcase how LCEL enables composable, testable, and declarative pipeline design using LangChain's core runtime features.
