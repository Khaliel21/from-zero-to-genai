# ?? LangChain Applications

This folder includes demos showcasing how to integrate LangChain with external tools and services.

## Contents

- `09_LangServe_demo.ipynb` – Deploying a LangChain agent using LangServe as an API endpoint.
- `10_LangGraph.ipynb` – Building multi-step workflows with LangGraph.
- `11_LangSmith.ipynb` – Adding observability and monitoring using LangSmith.

## Purpose

To explore advanced LangChain features for deploying, debugging, and tracking LLM-powered applications in production.
