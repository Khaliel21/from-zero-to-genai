# ?? Key Extraction

This folder contains notebooks focused on extracting key information and performing sentiment analysis on unstructured text data.

## Contents

- `02_Key_Data_Extraction.ipynb` – Extracts structured key data from raw textual inputs.
- `03_Sentiment_Analysis.ipynb` – Performs sentiment analysis to determine emotional tone or polarity.

## Purpose

To demonstrate how NLP techniques can identify critical information and analyze the tone of content from various sources.
