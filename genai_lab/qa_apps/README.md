# ? Question Answering Applications

This folder demonstrates question-answering systems built on top of structured and unstructured data sources.

## Contents

- `04_Qa_From_Sql.ipynb` – Q&A over relational databases using natural language queries.
- `05_Qa_From_Pdf.ipynb` – Q&A over PDF documents with embedded context-aware retrieval.

## Purpose

To implement retrieval-augmented generation (RAG) applications that answer user queries based on source documents or datasets.
